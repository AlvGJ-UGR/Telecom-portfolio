# 01 — Fundamentos SDR: Espectro en vivo + ADS-B

**Estado:** 🔵 Planeado

## Objetivo

Familiarizarme con SDR de forma práctica: visualizar espectro en tiempo real y decodificar señales ADS-B (1090 MHz) de aviones cercanos, con una antena colineal diseñada y fabricada por mí.

## Por qué importa

Es la base de todo lo demás en este repositorio: SDR es la herramienta central para el resto de proyectos (satélites, análisis de espectro, protocolos digitales).

## Enfoque

1. Configurar RTL-SDR con GNU Radio / SDR++.
2. Visualizar espectro en banda aeronáutica y compararlo con teoría de propagación.
3. Decodificar tramas ADS-B con `dump1090`, mostrar tráfico aéreo en mapa web en tiempo real.
4. Diseñar e imprimir en 3D un soporte para antena colineal (cable coaxial + tubo) sintonizada a 1090 MHz.
5. Medir la mejora de alcance con la antena casera vs. la antena stock del dongle.

## Material / BOM

- RTL-SDR Blog v3
- Cable coaxial RG58/RG6
- Soporte impreso en 3D
- Raspberry Pi o PC con Linux
